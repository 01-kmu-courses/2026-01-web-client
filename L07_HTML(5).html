웹 폼이란?
■사용자로부터 데이터를 입력받기 위한 HTML 요소들의 집합
■지금까지는 "보여주는" HTML → 이제 "입력받는" HTML
■로그인, 회원가입, 검색창, 설문조사 등에 사용
폼이 쓰이는 곳
🔐 로그인 — 아이디 / 비밀번호 입력
📝 회원가입 — 이름 / 이메일 / 성별 등
🔍 검색 — 검색어 입력 후 제출
📋 설문조사 — 선택지, 텍스트 입력
주요 폼 요소
태그역할
<form>폼 전체를 감싸는 컨테이너
<input>텍스트, 비밀번호, 체크박스 등
<select>드롭다운 목록
<textarea>여러 줄 텍스트 입력
<button>클릭 버튼
<label>입력 요소의 설명 텍스트
웹클라이언트컴퓨팅  4
form 기본 구조
■모든 폼 요소는 <form> 태그 안에 넣어야 함
■action  폼 데이터를 전송할 서버 주소 — 실습 시 # 으로 대체
■method  데이터 전송 방식 — GET 또는 POST
<form action="#" method="post">

  <label for="username">아이디</label>
  <input type="text" id="username"
    name="username">

  <label for="pw">비밀번호</label>
  <input type="password" id="pw"
    name="password">

  <button type="submit">로그인</button>

</form>
method설명
GET데이터가 URL에 붙어서 전송 — 검색에 주로 사용
POST데이터가 숨겨서 전송 — 로그인, 회원가입에 사용
💡 label 의 for 속성과 input 의 id 속성을 같게 맞춰야 연결됨
💡 서버 없이 실습할 때는 action="#" 으로 쓰면 오류 없이 새로고침만 됨

input 종류
■type 속성으로 입력 형태를 지정
type설명예시
text일반 텍스트이름, 아이디
password입력값 숨김 (●●●)비밀번호
email이메일 형식 검증이메일 주소
number숫자만 입력나이, 수량
tel전화번호 형식휴대폰 번호
date날짜 선택기생년월일
file파일 선택프로필 사진
hidden화면에 보이지 않음세션값 등
 
<input type="text"
  placeholder="이름 입력">

<input type="password"
  placeholder="비밀번호">

<input type="email"
  placeholder="example@email.com">

<input type="number"
  min="0" max="100">

<input type="date">

<input type="file">

input 주요 속성
■type 외에도 다양한 속성으로 입력을 제어할 수 있음
속성설명
placeholder입력 전 안내 텍스트 (흐리게 표시)
value기본값 설정
required필수 입력 항목으로 지정
disabled입력 불가 상태로 비활성화
readonly읽기 전용 — 수정 불가
maxlength최대 입력 글자 수 제한
min / max숫자/날짜의 최솟값 / 최댓값
 
<!-- 안내 텍스트 -->
<input type="text"
  placeholder="이름을 입력하세요">

<!-- 기본값 -->
<input type="text" value="홍길동">

<!-- 필수 입력 -->
<input type="email" required>

<!-- 비활성화 -->
<input type="text" disabled>

<!-- 최대 10글자 -->
<input type="text" maxlength="10">

checkbox
■여러 항목 중 하나 이상 선택할 때 사용
■각 항목이 독립적 — 여러 개 동시에 선택 가능
■checked 속성을 추가하면 기본으로 체크된 상태
<p>좋아하는 음식을 모두 선택하세요</p>

<input type="checkbox"
  id="pizza" name="food" value="pizza">
<label for="pizza">피자</label>

<input type="checkbox"
  id="burger" name="food" value="burger"
  checked>
<label for="burger">버거</label>

<input type="checkbox"
  id="pasta" name="food" value="pasta">
<label for="pasta">파스타</label>

좋아하는 음식을 모두 선택하세요

💡 같은 그룹은 name 속성을 동일하게 맞춤

<label for="male">남성</label>

<input type="radio"

<label for="female">여성</label>

select 드롭다운
■클릭하면 목록이 펼쳐지는 선택 상자
■선택지가 많을 때 radio 대신 사용
■selected 속성으로 기본 선택값 지정
■optgroup 으로 항목을 그룹으로 묶을 수 있음
<label for="city">거주 도시</label>
<select id="city" name="city">
  <option value="">-- 선택하세요 --</option>
  <option value="seoul">서울</option>
  <option value="busan">부산</option>
  <option value="daegu" selected>대구</option>
</select>

<!-- optgroup 으로 그룹화 -->
<select>
  <optgroup label="수도권">
    <option>서울</option>
    <option>인천</option>
  </optgroup>
  <optgroup label="영남권">
    <option>부산</option>
    <option>대구</option>
  </optgroup>
</select>
💡 selected 는 checkbox 의 checked 와 같은 역할
웹클라이언트컴퓨팅  10
textarea
■여러 줄의 텍스트를 입력받을 때 사용
■input type="text" 는 한 줄, textarea 는 여러 줄
■rows / cols 속성으로 기본 크기 지정
<label for="intro">자기소개</label>
<textarea
  id="intro"
  name="intro"
  rows="5"
  cols="40"
  placeholder="자기소개를 입력하세요">
</textarea>

<!-- 크기 조절 막기 -->
<textarea
  style="resize:none;"
  rows="4">
</textarea>
속성설명
rows세로 줄 수 (기본 높이)
cols가로 글자 수 (기본 너비)
resize:none크기 조절 비활성화
웹클라이언트컴퓨팅  11
button
■클릭 가능한 버튼을 만드는 태그
■type 속성으로 역할을 지정
<!-- 폼 제출 -->
<button type="submit">제출하기</button>

<!-- 폼 초기화 -->
<button type="reset">초기화</button>

<!-- 일반 버튼 (JS와 함께 사용) -->
<button type="button">클릭</button>

<!-- input으로도 버튼 만들기 -->
<input type="submit" value="제출">
<input type="reset" value="초기화">
type설명
submit폼 데이터를 서버로 전송
reset폼의 모든 입력값을 초기화
button기본 동작 없음 — JS로 제어
💡 button 과 input type="submit" 은 같은 역할 — button 이 더 많이 쓰임
